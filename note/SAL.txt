PROBLEMI RISCONTRATI E SOLUZIONI:

  (con src="/src/main.js") invece del file buildato da Vite (con gli asset hashati).
  CAUSA: Il workflow GitHub Actions (.github/workflows/deploy.yml) sembrava girare

  sistema Jekyll, che aveva deployato il sorgente grezzo invece della cartella dist/.

  automaticamente la cartella dist/ sul branch gh-pages. Su GitHub Pages è stata
  impostata la sorgente: "Deploy from branch → gh-pages → /(root)".
