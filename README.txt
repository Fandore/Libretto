Libretto v12 — pacchetto PWA

Come pubblicarla velocemente:
1. Carica tutti i file di questa cartella su un hosting statico HTTPS, per esempio GitHub Pages, Netlify o Vercel.
2. Apri il link /index.html da Safari su iPhone o Chrome su Android.
3. iPhone: Condividi > Aggiungi alla schermata Home.
4. Android/Pixel: menu Chrome > Installa app o Aggiungi a schermata Home.

Nota: i dati sono salvati nel localStorage del browser/dispositivo. Usa periodicamente Esporta backup JSON dall'app.
